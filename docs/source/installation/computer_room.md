# From computer room

1. Open *Anaconda Prompt*

2. Navigate to the disk where the environment was created
    ```sh
    X:
    ```

3. Activate **SISE** environment
    ```sh
    conda activate sise
    ```

4. Launch JupyterLab
    ```sh
    jupyter lab
    ```

5. Close JupyterLab  
    `CTRL-C`  
    `CTRL-C`
